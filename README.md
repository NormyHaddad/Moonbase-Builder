## Moonbase Builder  
  
#### Player controls:  
 - Movement: WASD  
 - Look around: Mouse  
 - RCS Jetpack up/down: Q/E  
 - Enter build mode: B  
 - Mine Ore: Right click  
  
#### Build mode camera controls:  
 - Movement: WASD  
 - Rotate: Q/E
